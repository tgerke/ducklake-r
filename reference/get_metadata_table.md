# Get a DuckLake metadata table

Get a DuckLake metadata table

## Usage

``` r
get_metadata_table(tbl_name, ducklake_name)
```

## Arguments

- tbl_name:

  Character string, name of the table to retrieve

- ducklake_name:

  Character string, name of the ducklake database

## Value

A DuckLake table of class `tbl_duckdb_connection`
