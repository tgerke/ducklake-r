# Test a DuckDB version string against the Quack minimum

Test a DuckDB version string against the Quack minimum

## Usage

``` r
quack_version_supported(version)
```

## Arguments

- version:

  A DuckDB version string such as `"v1.5.3"`.

## Value

`TRUE` if `version` is 1.5.3 or higher, otherwise `FALSE`.
